# 🐦 Chirp - Mini Rede Social

Uma aplicação web moderna desenvolvida com Laravel que permite aos usuários compartilhar pensamentos em formato de mensagens curtas (chirps), inspirada em redes sociais como o Twitter.

## 📸 Preview

> Interface simples, elegante e funcional para criação e interação com posts.

---

## ✨ Funcionalidades

- 🔐 Autenticação de usuários (login e registro)
- 📝 Criação de posts (chirps)
- 📄 Feed com listagem de mensagens
- ✏️ Edição de posts
- 🧾 Validação de formulários
- 🎨 Interface responsiva com Tailwind CSS

---

## 🧠 Conceitos aplicados

- Arquitetura MVC
- Rotas e Controllers no Laravel
- Blade Templates
- Autenticação
- CRUD completo
- Boas práticas de organização de código

---

## 🛠️ Tecnologias

- **PHP**
- **Laravel**
- **Blade**
- **Tailwind CSS**
- **MySQL**

---

## 📂 Estrutura do Projeto

| Arquivo | Descrição |
|--------|--------|
| `home.blade.php` | Feed principal com listagem de chirps |
| `login.blade.php` | Tela de login |
| `register.blade.php` | Cadastro de usuários |
| `edit.blade.php` | Edição de posts |
| `welcome.blade.php` | Página inicial |
